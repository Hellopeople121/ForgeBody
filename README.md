# ForgeBody

ForgeBody is a workout‑planning web application designed to help users build consistent fitness routines, track progress, and stay motivated. Created as a school project, it demonstrates full‑stack thinking, UI design, and modern web development practices.

---

## 🚀 Features

- **Custom Workout Plans** — Create and organize exercises for different days.
- **Exercise Library** — Add, edit, and categorize workouts.
- **Clean, Responsive UI** — Works on desktop and mobile.
- **Local Data Handling** — Saves user input without requiring an account.
- **Future‑Ready Structure** — Built so it can grow into a real product.

---

## 🛠️ Tech Stack

- **HTML / CSS / JavaScript**
- **Vite** (for fast development + bundling)
- **Modern UI components**
- **Environment variables** for secure API usage
